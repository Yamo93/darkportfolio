The Bookworm E-Shop

View Project Here!

This e-shop was created as part of a school project at the university. The task was to create a shop of some fashion and follow through the entire process from designing the product, develop it with code and finally deploy it to the Web.

I paid great attention to choosing the appropiate theme for this online bookshop, picking out convenient colors. My main designing tools were Adobe XD and some basic Photoshop editing.

A Mockup of the Bookshop

Since my mockups and graphics were neatly created, it was relatively easy to develop the user interface of the page with HTML and CSS. I used the SASS preprocessor to speed things up using the helpful features provided by SASS, such as using reusable mixins, writing nested styles and much more. I also spent quite a lot of time improving the JavaScript side of the page, providing dynamic functionality such as being able to add/remove items to a basket or a wishlist, checkout and so on.

The project is still open for further development. I'm planning on turning this site into a React-based application, and eventually use Node.js for the backend and be able to store items on a real database (document-based such as MongoDB).
