The Opinion Blog Portal

View Project Here!

This was initially a school project where I was given the task to create a blog portal with PHP for the server side and MySQL as a relational database management system.

I aimed to create a minimalistic and user-friendly blog portal while using simple features but still providing the user with the expected functionality of a typical blog portal.

Some of the implemented functionality on the page was:

  • User registration and authentication.
  • CRUD operations such as creating, reading, updating and deleting posts.
  • Creating bookmarks, comments and likes.
  • Uploading profile images.

The Comment Section

The site was implemented with HTML, CSS (including SASS and the BEM architecture methodology) as well as JavaScript where that was necessary. The site was made responsive and suitable for all kinds of devices.

I also used several Bootstrap components which was a big time saver and also made the responsive part easier to implement, since they are initially made to be responsive.
