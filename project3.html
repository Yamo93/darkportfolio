This was another school project where I was given the task to create a landing page for a company. I chose to create a page for a fictional web development agency which I gave the name "NÃ¤tbygg AB", simply translated to "Net Construction AB".

I planned the project by choosing a convenient theme, colors, fonts and images. I then planned the overall structure of the page and illustrated it in a sitemap seen below:

My Sitemap

I finally reached the point where it was time to start developing the site using basic HTML and CSS along with the SASS preprocessor. I tried to implement the BEM (Block-Element-Modifier) CSS architecture (as I usually do in my projects), mostly because it makes development much easier and also because it solves the specificity problem that often occurs when developing in CSS.
